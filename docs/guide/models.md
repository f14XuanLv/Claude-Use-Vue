# Claude各模型特性

## haiku
### 3 Haiku
- 最精简的上下文容量
- 可以识别和处理图片内容
- 能够读取 PDF 中的文字
- 无法识别 PDF 中的图片

### 3.5 Haiku 
- 纯文本模型
- 上下文容量比 Claude 3 Haiku 扩大了 2-4 倍
- 不支持多模态功能

## Sonnet
### 3.5 Sonnet 
- 支持完整的多模态能力
- 上下文容量20万token
- **可以识别 PDF 中的文字和图片**
- 在整体智能水平上领先其他版本

## Opus
### 3 Opus 
- 可以识别和处理图片内容
- 能够读取 PDF 中的文字
- 无法识别 PDF 中的图片
- **写作能力上可能超过 3.5 Sonnet**
